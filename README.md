# RBDA-17fall-Movie

## Dumbo HPC Environment:
**Java** 1.7.0_79

**Apache Spark MLlib** 1.6.0

**Apache Maven** 3.2.1

We used maven to build the project package:
> mvn archetype:generate 

for creating the project, and

> mvn package 

for building the target JAR file
